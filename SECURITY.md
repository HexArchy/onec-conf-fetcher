# Security Policy

## Scope

This tool is designed for **authorized penetration testing** and security assessments only. Use it responsibly and only on systems you have explicit permission to test.

## Reporting Vulnerabilities

If you find a security issue in this module (e.g., credential leakage, unintended data exposure), please report it privately:

1. **GitHub Security Advisories**: [Create a private advisory](https://github.com/HexArchy/onec-conf-fetcher/security/advisories/new)
2. **Email**: nikitabelekov@gmail.com

Do not open public issues for security vulnerabilities.
